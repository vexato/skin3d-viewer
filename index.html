Documentation - Skin 3D

Ajouter le Composant Skin 3D

Pour intégrer le composant Skin 3D dans votre thème Azuriom, choisissez l'une des directives suivantes selon le type de service que vous souhaitez utiliser :

Service Premium

Utilisez cette directive pour activer le service premium :

                @include('skin3d::components.skin3d', ['service' => 'premium'])

Service Skin-api

Si vous préférez utiliser le service Skin-api, appliquez la directive suivante :

                @include('skin3d::components.skin3d', ['service' => 'skin_api'])

Instructions d'Intégration

  1. Insérez la directive choisie (premium ou skin_api) à l'endroit souhaité dans votre code.
  2. Enregistrez vos modifications et vérifiez le rendu sur votre site.

Remarques

- Assurez-vous que le plugin Skin3D Viewer est bien installé et configuré sur votre site.
- Pour utiliser le service Skin-api, consultez le plugin Skin-api.
- Pour toute question ou problème, consultez la documentation officielle d'Azuriom ou contactez le support.
